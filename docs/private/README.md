# Private documentation

This folder is for documentation you want **backed up in Git** but **not published** on the live site (Vercel).

- Add your SQL structure document here (e.g. `sql-structure.md` or `schema.sql`).
- Files here are committed and pushed to the repo.
- They are excluded from the Vercel deployment via `.vercelignore`, so they are not available on the public site.
